# Neovim Cheatsheet

An interactive, searchable Neovim cheatsheet: keybindings, motions, and editing commands on one page, with live search and click-for-detail descriptions.

**Live site: [neovimcheatsheet.com](https://neovimcheatsheet.com)**

## Features

- Live search across every binding, with match highlighting
- Click any binding for a description and example
- Remappable bindings you can edit in the browser, with JSON import/export
- Light and dark mode
- No tracking, no logins, no telemetry

## Contributing

Corrections and additions are very welcome. Accuracy is the priority here, so **please cite a source** (the official Neovim docs or `:help`) for any change. See [CONTRIBUTING.md](CONTRIBUTING.md) for the quick guide.

If this cheatsheet saves you a trip to the docs, a ⭐ helps others find it. Corrections and PRs are always welcome.
